## 📌 About the Project
This project is a custom-designed, 3D-printed fume extractor system aimed at removing harmful gases and smoke generated during soldering operations. The project involved electronic component selection, aerodynamic enclosure design from scratch using SolidWorks, and rapid prototyping.

## ⚙️ Technologies & Tools
* **CAD Design:** SolidWorks
* **Manufacturing (Prototyping):** Creality Ender 3 V3 CoreXZ (FDM 3D Printing)
* **Electronics:** 12V 120mm Case Fan, PWM DC Motor Speed Controller Module, 12V Power Supply Adapter

## 🛠️ 3D Design (CAD)
The enclosure was modeled as a solid body in SolidWorks, with specific tolerances to ensure a perfect fit for the fan and to maximize airflow efficiency.

## 🖨️ Manufacturing & Assembly
After finalizing the design, the parts were exported as STL files and 3D printed using a Creality Ender 3 V3 CoreXZ. Following the enclosure production, the electronic components were soldered, wired, and fully integrated into the system to create a working prototype.

## 📂 Repository Structure
* `CAD_Files/` : SolidWorks part and assembly files (.sldprt, .sldasm)
* `STL_Files/` : 3D printable models (.stl)
